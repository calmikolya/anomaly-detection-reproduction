# Anomaly Detection — воспроизведение 5 моделей на MVTec AD

Этот репозиторий — практическое воспроизведение пяти известных подходов к anomaly
detection (поиску дефектов на изображениях промышленных объектов): **PatchCore, PaDiM,
STFPM, SimpleNet и DRAEM**. Эти пять выбраны не случайно — они представляют разные идеи
решения одной задачи (сравнение с эталонными признаками, статистика, дистилляция
teacher-student, синтетические аномалии в признаках, синтетические аномалии на
пиксельном уровне) и вместе дают неплохую картину того, как вообще устроен anomaly
detection последних лет. Задача была не просто прочитать про эти методы, а прогнать
каждый своими руками на датасете **MVTec AD** (класс `bottle`) и посмотреть, насколько
цифры из статей воспроизводятся в реальности — да ещё и на обычном Mac без видеокарты.

## Структура репозитория

Каждая модель — это клон оригинального репозитория автора с точечными правками под
CPU/macOS (архитектуру не трогали). Общего раннера нет — у каждой модели свой способ
запуска, как и было у авторов, и мы не стали городить общую обвязку поверх пяти совсем
разных кодовых баз, чтобы не сломать чужую логику.

```
.
├── results.md                  # метрики всех 5 моделей + сравнение с оригинальными статьями
├── patchcore-inspection/        # 1. PatchCore (amazon-science/patchcore-inspection)
├── padim_run/                   # 2. PaDiM (через библиотеку anomalib)
├── STFPM/                       # 3. STFPM (gdwang08/STFPM)
├── SimpleNet/                    # 4. SimpleNet (DonaldRR/SimpleNet)
└── DRAEM/                        # 5. DRAEM (VitjanZ/DRAEM)
```

## Conda-окружения

Вместо пяти изолированных окружений (диска было жалко) обошлись двумя общими:

- **`patchcore`** — для PatchCore, STFPM, SimpleNet и DRAEM
- **`anomalib`** — отдельно для PaDiM, потому что у библиотеки `anomalib` своё большое
  дерево зависимостей (pytorch-lightning, kornia и т.д.), которое проще не смешивать
  с остальным

```bash
conda activate patchcore   # для моделей 1, 3, 4, 5
conda activate anomalib    # для модели 2
```
